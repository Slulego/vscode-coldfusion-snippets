# vscode-coldfusion-snippets
A humble collection of commonly used ColdFusion tags.

Available Tags:

-	&lt;cfapplication&gt;
-	&lt;cfdump&gt;
-	&lt;cfif&gt;
-	&lt;cfif else&gt;
-	&lt;cfinclude&gt;
-	&lt;cflocation&gt;
-	&lt;cfloop&gt; (index)
-	&lt;cfoutput&gt;
-	&lt;cfquery&gt;
-	&lt;cfset&gt;
-	&lt;cfscript&gt;
